# User stories

As a User I want to be able to:
- create an account to access the features of the application and be identified by my unique username and email address.
- log in to my account using my email address and password so that I can access my personalized content and settings.
- reset my password in case I forget it, so that I can regain access to my account.
- create new projects and manage them, so that I can organize my work and collaborate with others.

Project administrator I want to be able to:
- invite users to join my project, so that I can collaborate with them and share resources.
- assign roles and permissions to users in my project, so that I can control access to different features and data.
- view and manage the activity logs of my project, so that I can monitor the actions of users and ensure the security of my project.
- generate reports on the progress and performance of my project, so that I can evaluate the success of my project and make informed decisions.
- customize the settings and preferences of my project, so that I can tailor the project environment to my needs and preferences.
- archive or delete my project when it is completed or no longer needed, so that I can keep my workspace organized and free of clutter.
- remove users from my project if necessary, so that I can maintain a productive and collaborative environment.

As a Project member I want to be able to:
- create and edit tasks within my project, so that I can contribute to the completion of the project goals.
- comment on tasks and communicate with other members, so that I can collaborate effectively and share feedback.
- change status of tasks to indicate their progress, so that I can keep track of the work being done and identify any bottlenecks.
- change asignee, estimate and due date of tasks, so that I can manage my workload and meet deadlines.
- view the overall progress of the project, so that I can understand how my work contributes to the larger goals of the project.   
- receive notifications about updates and changes in the project, so that I can stay informed and respond promptly to any issues or requests.
- access the project resources and documentation, so that I can find the information I need to complete my tasks effectively.
